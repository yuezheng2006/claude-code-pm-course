# 面向产品经理的Claude Code课程

一个交互式课程，教导产品经理如何有效使用Claude Code进行日常工作。通过SingTech（虚拟K歌娱乐公司）的真实PM工作场景进行实践学习。

## 课程结构

### 模块0：入门指南（约30分钟）
- **0.0 介绍** - 课程概览和您将学到的内容
- **0.1 安装** - 安装Claude Code和前置条件
- **0.2 启动 & 克隆** - 启动Claude Code并克隆本课程

### 模块1：基础知识（约2小时）
- **1.1 欢迎** - SingTech介绍和课程项目（10分钟）
- **1.2 可视化文件** - 设置文件可视化工具进行分屏工作流程（15分钟）
- **1.3 第一个任务** - 读取文件、转换内容和使用输出样式（25分钟）
- **1.4 代理** - 克隆Claude并行工作（20分钟）
- **1.5 自定义子代理** - 创建具有自定义角色的专业代理（15分钟）
- **1.6 项目记忆** - 使用CLAUDE.md作为项目上下文（15分钟）
- **1.7 Claude Code导航** - 导航技能和键盘快捷键（15分钟）

### 模块2：产品经理工作流（约1.5小时）
- **2.1 撰写PRD** - 创建产品需求文档（30分钟）
- **2.2 分析数据** - 处理数据和数据分析（25分钟）
- **2.3 产品策略** - 战略规划和优先级排序（30分钟）

## 如何使用本课程

### 交互式学习路径（推荐）
1. 克隆此仓库
2. 进入课程材料目录：`cd course-materials`
3. 启动Claude Code：`claude`
4. 运行 `/start-1-1` 开始第一课
5. 跟随指导性的动手练习

### 参考学习路径
访问 [课程网站](https://ccforpms.com) 查看每个模块的详细参考指南，可作为快速查找文档使用。

## ⚠️ 重要：Claude Code使用说明

**当您首次在Claude Code中打开此仓库时：**
- ❌ **请勿安装npm依赖** 或运行 `npm install`
- ❌ **请勿构建网站** 或运行 `npm run build`
- ❌ **请勿在未被要求时进行任何设置更改**

**而是：**
- 📖 按照指导阅读课程内容
- 🎯 等待每个模块中的明确指示
- 💬 询问用户下一步想要做什么

本课程是自包含的 - 我们只会在需要时处理设置。您的任务是交互式地引导用户完成学习体验。

## 您将学到什么

**Claude Code核心能力：**
- **文件操作** - 直接从终端读取、写入和编辑文档
- **并行代理** - 启动多个Claude实例来同时处理文件
- **自定义子代理** - 为不同视角创建专门的评审员
- **项目记忆** - 使用CLAUDE.md维护持久上下文
- **@-提及** - 引用文件和文件夹为Claude提供完整上下文
- **图像分析** - 粘贴截图和原型获取即时反馈

**应用于真实PM工作：**
- 处理会议记录并提取行动项
- 从多个来源综合用户研究
- 使用多视角反馈撰写PRD
- 分析产品数据并运行实验
- 制定竞争策略并创建演示文稿

## 前置条件

- Claude Code访问权限（需要Claude账户）
- Mac、Windows或Linux计算机
- 基本产品管理经验（知道PRD是什么）
- 愿意通过实践学习

**几乎无需编码经验。无需终端经验。** 如果您能用自然语言描述您想要什么，您就可以使用Claude Code。

## 关于本课程

本课程教导PM如何将Claude Code作为思考伙伴使用，而不仅仅是自动化工具。您将学到：
- 在不牺牲质量的前提下工作得更快
- 即时获得多视角反馈
- 高效处理研究和数据
- 使用AI协助撰写更好的文档

**完成时间：** 完整交互式学习路径约3.5小时（模块1约2小时，模块2约1.5小时）
