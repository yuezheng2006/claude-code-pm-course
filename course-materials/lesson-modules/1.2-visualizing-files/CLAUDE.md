## 常见学生问题和答案

**问:"我必须使用 VS Code/Obsidian 吗?"**
答:"不需要!你可以使用任何本地文本编辑器或文件管理器。VS Code、Obsidian、Cursor 都是很棒的选择。使用你觉得舒服的任何工具!"

**问:"我可以使用不同于分屏的布局吗?"**
答:"当然可以!有些人喜欢:
- 终端全屏,用 Alt+Tab 切换到编辑器
- 三分屏(终端、编辑器、浏览器)
- 双显示器(一个显示终端,另一个显示编辑器)

使用对你有效的任何方式!关键是能够看到 Claude 正在创建的文件。"

**问:"为什么有些编辑器看不到 .claude/ 文件夹?"**
答:"因为 .claude/ 是隐藏文件夹(以点开头)。在大多数编辑器中都可以设置显示隐藏文件:
- VS Code: 默认显示
- Obsidian: 设置 > 文件与链接 > 显示隐藏文件
- 文件管理器: 查看选项中启用显示隐藏文件"

**问:"如果我不想使用复杂的编辑器怎么办?"**
答:"你可以直接使用系统的文件管理器(Mac Finder 或 Windows 文件资源管理器)。虽然功能简单,但至少能看到文件结构。完成后可以随时升级到更强大的工具。"

**问:"测试文件没有出现。我该怎么办?"**
答:"让我们排查:
1. 你在本地工具中查看吗?(记住 Claude Code 在本地文件系统中工作)
2. 你在正确的文件夹中查看吗?(lesson-modules/1.2-visualizing-files/)
3. 尝试刷新文件浏览器或重新打开编辑器
4. 让我检查文件是否存在:[运行 ls 命令]

**重要:** Claude Code 直接在本地文件系统中创建文件。确保你在本地工具中查看。

我们会一起解决的!"
