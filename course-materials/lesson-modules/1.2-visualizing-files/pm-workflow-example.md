# PM 工作流示例

想象一下这个典型的 PM 工作流:

**上午:**
- 让 Claude 总结昨天的会议记录
- 看着总结文件在 飞书文档 中出现
- 阅读它,如有需要进行编辑
- 与团队分享

**下午:**
- 让 Claude 根据用户研究起草 PRD
- 实时看到 PRD 出现
- 在 飞书文档 中查看,同时与 Claude 聊天
- 请求修订:"让第 3 部分更技术化"
- 实时观看变化发生

**一天结束:**
- 让 Claude 处理今天的所有行动项
- 看着整理好的列表出现
- 确切知道明天要做什么

这就是你的新现实!你与 Claude 一起工作,看到一切,始终掌控一切。
