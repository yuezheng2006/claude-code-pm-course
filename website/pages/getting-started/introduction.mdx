## 0.0: 介绍

## 📖 概述

Claude Code 是一个 AI 助手，可以直接在你的电脑上处理你的文件和文件夹。与传统聊天机器人不同，你需要手动复制粘贴上下文到其中，Claude Code 通过自然对话直接读取你的文档、分析数据并创建材料。

**关键要点：** Claude Code 将产品经理从手动执行者转变为编排者。你专注于战略决策，而 Claude 处理文档合成、数据分析和工作流程自动化。


## 🎯 你将构建什么

本课程教你使用 Claude Code 构建一个产品经理生产力系统。

**核心能力：**
1. **文件操作** - 使用 @ 符号读取、写入和引用文件
2. **视觉分析** - 粘贴截图并分析模型、图表、数据
3. **并行代理** - 同时运行多个 AI 实例进行批量处理
4. **自定义子代理** - 具有个性的永久专业团队成员
5. **项目记忆** - 通过 CLAUDE.md 持久存储关于你的产品和团队的上下文


## 📋 要求

- Claude Pro 或 Max 订阅（每月 20 美元）
- Mac、Windows 或 Linux 电脑
- 安装需要 15 分钟

**时间投入：**
- 级别 1（基础）：2-3 小时
- 级别 2（实践者）：4-5 小时
- 级别 3（大师）：3-4 小时 [即将推出]

总计：10-12 小时成为高级用户。每周节省 5-10 小时。


## 💼 真实的 PM 用例

### 战略工作
- **PRD 编写** - 将研究、访谈和竞争分析合成到连贯的文档中
- **路线图规划** - 分析市场数据和利益相关者输入
- **客户反馈分析** - 在几分钟内处理数百个响应
- **竞争情报** - 生成 SWOT 分析和比较矩阵

### 执行与组织
- **工单生成** - 将功能规范转换为用户故事和 Jira 工单
- **多格式转换** - 将 PRD 转换为单页文档、演示文稿、发布说明
- **文件组织** - 结构化研究文件、会议记录、截图
- **数据编辑** - 在不打开 Excel 的情况下操作电子表格和 CSV

### 沟通
- **利益相关者更新** - 起草邮件、企业微信消息、飞书文档页面
- **发布说明** - 将工程变更日志转换为面向客户的语言
- **演示文稿叙述** - 从战略文档构建幻灯片演示文稿
- **知识库** - 在会话间维护持久的产品知识

### 研究与分析
- **SEO 审计** - 分析营销页面和内容
- **竞争对手研究** - 创建比较表格和功能矩阵
- **洞察合成** - 结合用户访谈、调查、支持工单、应用评价
- **网络研究** - 自动化的竞争情报收集


## 🔄 Claude Code vs 传统聊天机器人

| 功能 | 传统聊天机器人 | Claude Code |
|------|----------------|-------------|
| **文件访问** | 无 | 在你的电脑上读取/写入任何文件 |
| **记忆** | 会话后消失 | 永久项目记忆 (CLAUDE.md) |
| **执行** | 建议，但你来做 | 实际执行命令并创建文件 |
| **多文件** | 一次性响应 | 同时跨数十个文档工作 |
| **代理** | 单个实例 | 并行运行多个代理 |
| **集成** | 有限 | 通过 MCP 连接到飞书文档、Linear、企业微信、GitHub |
| **版本控制** | 无历史记录 | Git 集成、检查点、回滚 |
| **自动化** | 手动复制粘贴 | 通过自定义命令实现真正自动化 |

**区别：** 持久性和执行。Claude Code 不仅仅提供建议 - 它与你一起完成工作。


## 📊 真实例子：Monday.com PM

Monday.com 的一位产品经理使用 Claude Code 收集并分析了关于他产品的 34,000 条 Reddit 评论。Claude 识别了模式、痛点和优先主题，这些直接影响了路线图。

**影响：** 几周的手动工作 → 几个小时。这不是要取代 PM 判断力 - 而是处理繁重工作，让你专注于战略思考。
