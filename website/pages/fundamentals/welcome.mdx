## 1.1: 欢迎来到 SingTech

 - **完成时间：** 20 分钟
 - **前提要求：** 已安装 Claude Code


> **在 Claude Code 中启动此模块：** 运行 `/start-1-1` 来开始互动体验。

## ▶️ 开始使用提醒

⚠️ **如果你还没有克隆仓库：** 请先回到 [模块 0.2：启动 Claude Code 并克隆课程](/getting-started/start-and-clone)。你需要在电脑上安装课程文件才能继续。


**在 Claude Code 中启动本课程有两种方式：**

### 选项一：在文件资源管理器中查找

导航到你的文档文件夹，找到 `claude-code-pm-course` 文件夹。然后右键点击并选择"在文件夹中新建终端"

看起来像这样：

![从文件资源管理器打开文件夹](/images/folder-location.png)

然后输入 `claude` 并按回车

### 选项二：从终端打开

在终端中输入这些命令：
```bash
cd ~/Documents/claude-code-pm-course
claude
```

这会导航到你的课程文件夹并直接在其中打开 Claude Code。

**无论哪种方式，你都会到达同一个地方。然后，要启动这个模块：**

> **在 Claude Code 中启动这个模块：** 运行 `/start-1-1` 来开始互动体验。

这就是你启动每个模块的方式！

## 📖 概述

这个模块介绍 SingTech（你的虚拟公司）并解释课程结构。

**关键要点：** 这是一个动手实践课程，你将使用 Claude Code 进行真实的 PM 工作。


## 🎓 课程如何运作

### 两个部分

**1. 互动模块（你现在正在做的）**
- 使用 Claude Code 完成模块
- 每个模块以斜杠命令开始：`/start-1-1`、`/start-1-2` 等
- Claude 一步步指导你完成练习
- 你实际执行工作（不仅仅是阅读）
- 为 SingTech 进行真实的 PM 任务

**2. 参考指南（你现在正在阅读的）**
- 每个模块的综合独立指南
- 可以在网站上阅读，无需使用 Claude Code
- 当你需要更深入的解释时使用
- 非常适合复习和后续参考

### 课程结构

- **模块 1：基础知识** - 核心概念和工作流
- **模块 2：高级内容** - 真实的 PM 工作（PRD、数据分析、策略）

## 🏢 什么是 SingTech？

### 公司概述

**SingTech(星歌科技)** 是为本课程创建的一个虚构的K歌娱乐科技公司。想象一下 **全场景K歌解决方案平台，提供软硬件结合的全场景K歌解决方案。**

**公司详情：**
- **成立时间：** 2010 年
- **阶段：** B 轮创业公司
- **融资：** 数千万人民币级别
- **团队规模：** 80 名员工（35 名产研，45 名 GTM）
- **指标：** 年营收过亿RMB，80万月活用户，快速增长

**你的角色：** 高级产品经理
- 你负责移动端与云服务产品管理
- 向 产品VP（产品VP A）汇报
- 与 CEO A、CTO A 密切合作

**使命：** 为不同场景的用户提供专业K歌娱乐体验

**产品：** K歌娱乐平台，结合曲库管理、演唱录制、社交分享和AI技术，专为多场景K歌打造

**关键差异化：** 比全民K歌更专业音质，比传统KTV更智能化，软硬件全栈结合

### 👥 用户角色

- **KTV运营者 A - 商业KTV老板：** 拥有3家中型KTV的连锁经营者，专注于系统稳定性、曲库更新和经营效率

- **家庭用户 A - K歌爱好者：** 35岁家庭主妇，喜欢周末在家K歌，专注于音质效果、操作简便和家庭娱乐

- **年轻用户 A - 移动K歌用户：** 22岁大学员，经常用手机K歌，专注于新歌更新、AI修音和社交互动


## ⏱️ 时间投入

**每个模块：** 15-30 分钟
**每个级别：** 2-4 小时
**完整课程：** 20-30 小时

**但是：** 你可以按照自己的节奏进行！
- 一次完成一个模块
- 在级别之间休息
- 随时回来（进度会被保存）

### 你将看到的内容

在整个课程中，你会看到 Claude 运行类似这样的命令：

```
⏺ Read(company-context/COMPANY.md)
  ⎿  Read 45 lines

⏺ Bash(ls -la company-context/)
  ⎿  Listed 4 files
```

**不用担心理解这些！** 这些是 Claude 在后台工作。你只需要专注于 PM 工作。

### 权限提示

偶尔，你会看到询问的提示：
你想允许这个命令吗？

**只需点击：** 是，不要再询问

这让 Claude 能够顺畅工作，而不必每次都询问权限。

总共 15-25 小时，按照你自己的节奏。

是的！不需要编码。

绝对可以。完成模块 1 后，你可以将这些模式应用到你的实际 PM 工作中。
