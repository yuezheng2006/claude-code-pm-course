## 1.2: 使用飞书可视化文件

 - **完成时间：** 15-20 分钟
 - **先修要求：** 模块 1.1 完成，基本熟悉文件系统


> **在 Claude Code 中开始此模块：** 运行 `/start-1-2` 开始互动体验。

## 📖 概述

设置飞书文档以在 Claude Code 旁边实时可视化您的项目文件，实现无缝的分屏工作流程。

**关键要点：** 可视化工作区让您观看文件在 Claude 工作时出现和更新。


## 🎯 为什么可视化很重要

在终端中使用 Claude Code 感觉像是在盲目工作。您要求 Claude 创建文档，但您看不到正在发生什么。

**解决方案：** 在终端旁边打开 Obsidian。现在您可以：
- 观看 Claude 创建文件时的文件出现
- 在 Claude 工作时阅读文档
- 验证一切是否正确组织
- 始终确切知道正在发生什么

这将 Claude Code 从一个神秘的黑箱转换为一个透明的工具，您可以实时观看和验证。


## 🎨 为什么选择飞书？

飞书文档非常适合产品经理，因为它是为协作文档（PRD、笔记、文档）构建的，具有干净、团队友好的界面。它是您的公司标准工具，具有出色的 markdown 支持。

**替代方案：** VS Code（更强大）或 Cursor（VS Code + AI）也能很好地工作。我们将在本课程中使用飞书文档，因为它是您的公司标准工具。


## 🚀 设置指南

**访问：** 访问 https://www.feishu.cn/ 并使用您的公司账户登录

**连接您的项目：** 在飞书文档中，选择"从文件夹导入"并导航到您的课程文件夹（如果需要，请询问 Claude 路径）

**分屏工作流程：** 将 Claude Code 终端和飞书文档并排放置，以实时观看文件同步


## ⚠️ .claude/ 文件夹

**关键限制：** 飞书文档无法显示隐藏文件夹（以 `.` 开头的文件夹）

`.claude/` 文件夹包含代理、命令和设置。您不会在飞书文档中看到它 - 这是正常的。

**查看 `.claude/` 文件的方法：**
- **Mac：** 在 Finder 中按 `Cmd + Shift + .` 显示隐藏文件夹
- **Windows：** 在文件资源管理器查看选项卡中启用"隐藏的项目"

**何时需要它：** 模块 1.5（自定义子代理）及以后。现在，专注于常规项目文件。

**注意：** Claude Code 可以访问 `.claude/` 文件 - 只有飞书文档无法显示它们。


## 🎬 实时可视化演示

当 Claude 创建或编辑文件时，您将立即在飞书文档中观看它们同步。

**试试看：**

1. 使用您的项目文件夹打开飞书文档
2. 在 Claude Code 中询问：*创建一个名为 test-visualization.md 的文件并添加一些内容*
3. 观看飞书文档 - 文件立即同步
4. 点击它阅读 Claude 写的内容

就是这样！无需刷新。文件在 Claude 工作时实时同步。

**要做的：**
- 持续使用分屏工作流

**不要做的：**

## 🐛 故障排除

**我看不到任何文件：**

**可能原因：** 连接了错误的文件夹

**修复方法：**
1. 询问 Claude：我的课程文件夹路径是什么？
2. 在飞书文档中：断开连接并重新导入
3. 再次导入文件夹，使用 Claude 给的确切路径
4. 你的文件现在应该会出现

**常见错误：** 打开了父文件夹而不是项目文件夹本身

**示例：**
- ❌ 打开了：`/Users/yourname/Documents`（层级太高）
- ✅ 应该打开：`/Users/yourname/Documents/claude-code-pm-course`

### 文件没有在飞书文档中更新

**可能原因：** 飞书文档需要检测到更改

**修复方法：**
1. 点击飞书文档窗口外部然后回到窗口（强制刷新）
2. 或者：关闭并重新打开特定文件
3. 或者：重启飞书文档（罕见，但有效）

**预防：** 通常不会发生。如果反复出现，检查：
- 飞书文档是否连接到正确的文件夹？
- 你是否在编辑正确的文件？（检查文件路径）

### 我在文件中看到奇怪的代码/格式

**可能原因：** 你处于编辑视图而不是阅读视图

**你看到的内容：**
```markdown
# 这是一个标题

**这是粗体** 和 *这是斜体*

- 项目符号
- 另一个项目符号
```

**你应该看到的内容：** 格式化的文本（实际的标题、实际的粗体、实际的项目符号）

**修复方法：**
- 在飞书文档中切换到预览/阅读模式
- 或者点击文档顶部的预览按钮

### .claude/ 文件夹在哪里？我找不到它！

**答案：** 它是隐藏的！这是正常且预期的。

**要查看它：**

**Mac：**
1. 打开 Finder
2. 导航到你的项目文件夹
3. 按：`Cmd + Shift + .`（句点）
4. `.claude/` 变得可见（变暗/灰色）

**Windows：**
1. 打开文件资源管理器
2. 导航到你的项目文件夹
3. 查看选项卡 → 选中隐藏的项目
4. `.claude/`
